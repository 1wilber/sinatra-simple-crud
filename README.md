# Simple sinatra CRUD

Ejemplo de como crear un CRUD con sinatra, totalmente desde cero, siguiendo una estructura parecida a la de rails

![](./preview.png)

## Creado con

* Sinatra
* Sinatra-flash
* sinatra-activerecord
* Rake
* sqlite3
* pry

## Como empezar

```rb
$ bundle install && ruby app.rb
```
Servidor alojado en el __localhost:4567__
